window对象属性

document 对话框显示当前文本
frames 当前对话框所以frame对象集合
location 指定当前文档URL
name 对话框名字
status 状态栏中的当前信息
defaultStatus 状态栏中的当前信息
top 最顶层的浏览器对话框
parent 当前对话框的父对话框
opener 打卡当前对话框的父对话框
closed 当前对话框是否关闭逻辑值
self 当前对话框
screen 用户屏幕，提供屏幕尺寸，颜色深度信息
navigator 获取与浏览器相关信息

window对象方法

alert() 弹出一个警告对话框
confirm() 确认对话框中显示指定字符串
prompt() 弹出提示对话框
open() 打开新浏览器并显示URL或名字引用的文档，并设置创建对话框属性
close() 关闭被引用对话框
focus() 被引用的对话框放在所有窗口前面
blur() 被引用的对话框放在所有窗口后面
scrollTo(x,y) 把对话框滚动到指定坐标
scrollBy(offsetx,offsetx) 安装指定位移滚动对话框
setTimeout(timer) 在指定数毫秒后，对传递的表达式求值
setlnterval(interval) 指定周期性执行代码
moveTo(x,y) 将对话框移动到指定坐标
moveBy(offsetx,offsety) 将对话框移动到指定的位移量处
resizeTo(x,y) 设置对话框大小
resizeBy(offsetx,offsety) 按照指定的位移量设置对话框的大小
print() 相当于浏览器中的打印按钮
navigate(URL) 将对话框显示URL指定页面
