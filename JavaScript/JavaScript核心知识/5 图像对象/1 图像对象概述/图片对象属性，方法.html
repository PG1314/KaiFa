图像对象属性

border 返回或设置图片的边框宽度，以像素为单位
height 返回或设置图片的高度，以像素为单位
hspace 返回或设置图片左边和右边的文字与图片之间的间距，以像素为单位
lowsrc 返回或设置代替图片的低分辨率图片的URL
name 返回或设置图片名称
src 返回或设置图片的URL
vspace 返回或设置图片上下面的文字与图片的间距，以像素为单位
width 返回或设置图片的边框宽度，以像素为单位
alt 返回或设置鼠标经过图片时显示文字
complete 判断图像是否完全被加载，如果被完全加载，属性返回true值

图像对象的事件

abort 当用户放弃加载图片时触发该事件
load 成功加载图片时触发事件
crror 在加载图片过程中产生错误时触发该事件
click 图片上点击时触发事件
dblclick 图片上双击鼠标触发事件
mouseover 鼠标移动到图片时触发事件
mouseout 鼠标从图片移开时触发事件
mousedown 图片上按下鼠标时触发事件
mouseup 图片上释放鼠标时触发事件
mousemove 图片上鼠标移动时触发事件
