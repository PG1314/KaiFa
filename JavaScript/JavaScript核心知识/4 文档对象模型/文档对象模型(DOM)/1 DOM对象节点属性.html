DOM 概述

Document 文档 DOM根据网页创建文档对象
Object 对象 对象是一种对立的数据集合
Model 模型 将文档对象表示未树状模型

根节点：最顶层的节点，称为根节点
父节点：一个节点上的节点为该节点的父节点
子节点：一个节点下的节点为该节点的子节点
兄弟节点：多个节点在同一层，这几个节点称兄弟节点
后代节点：一个节点下的节点下的节点为该节点的子节点
叶子节点：树状图结构下最低部节点为叶子节点
文档模型中节点有三种：
  • 元素节点：如：body,p,a等标记，组成的文档模型语义逻辑结构

  • 文本节点：包含元素节点内容如：p标记的文本
  • 属性节点：元素节点属性：如：a标签的href属性等，大部分属性节点隐藏浏览器背后不可见
